# Security Posture Open Work

Temporary scratch plan for unresolved security items that are not yet covered by
an issue or a durable project document.

## 1. Repo-Specific Secret Pattern Watch

Open decision:

- Decide whether project-specific secret formats need custom detection
  patterns.

Planning constraints:

- Add custom patterns only after evidence of a missed project-specific secret
  shape.
- Prefer narrowly scoped patterns with tests, ownership, and review guidance.
- Avoid duplicate scanner noise; do not add another secret-scanning tool
  without a concrete gap and failure policy.
