# Global sidnavigering för innehållsutrymme

Status: Antagen 2026-06-21.

Applikationen använder global sidnavigering för att prioritera vertikalt
utrymme för sidornas innehåll, tabeller och arbetsytor. Sidnavigeringen ligger
till vänster så att den inte konkurrerar med flytande åtgärdsverktyg på höger
sida, och den kan fällas ihop så återkommande användare får en kompakt
ikonbaserad navigering.

På små skärmar används en sidolåda som öppnas från en flytande menyknapp. Det
ger samma navigeringsstruktur utan att lägga en permanent navigationsyta över
innehållet.
